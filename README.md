Planteamos de la aplicacion:

A los clientes (usuarios) registrarse, iniciar sesión y realizar pedidos.
A los administradores gestionar productos y supervisar las órdenes. 

    Actualmente no existe un sistema digital, por lo que se requiere desarrollar una aplicación web completa que simule el flujo real de pedidos usando una API falsa con JSON Server.

    El diseño visual, estructura de pantallas y componentes UI ya están definidos en Figma (login, menú, órdenes, dashboard admin).
